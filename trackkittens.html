OVERVIEW

Where are
kittens today?

Track Kittens is a mobile app for monster tracking, designed for entertainment purposes. The idea is that many cats wandering on the street, and as a cat lover, the user can find them, mark their locations, arrest them, and save them.
This app has a user system that allows multiple users to register, login and logout as they wish. Once logged in, users could search, add or delete cats as they wish in their own kitten lists, and edit the kittensâ€™ profiles. On the other hand, users could update or add the new location of each cat.
